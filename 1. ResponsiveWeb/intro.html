1. What is Frontend?

Frontend web programming

ユーザが、直接に操作する画面を構築することです

2. What can frontend do?

  • Web Application Development
  • Mobile Application Development
  • Desktop Application Development
  • Game Development
  • Data Analysis and Visualization

3. What does frontend workflow look like?

  • User Experience Design, by UX Designer
  • User Interface Design, by UI Designer
  • Frontend Implementation, by Frontend Engineer

4. What languages do we use to code frontend?

  • HTML (HyperText Markup Language): ウェブページの文を意味と役割を付ける言語です
  • CSS (Cascading Style Sheets): ウェブページのスタイルを指定するための言語です
  • Javascript: ウェブページに動きや対話性を付加するための言語です

5. What are sass, less, coffeescript, etc.?

  • HTML: .pug, .erb, .php
  • CSS: Sass/Less, stylus
  • Javascript: coffeescript, typescript
  • Browser can only understand HTML, CSS and Javascript, all these languages above will be compiled down to HTML, CSS, and Javascript.

  • Manipulate Web Elements with plain Javascript
  • Use Dom Events to create user action logic
  • Make Http Request to backend and update webpage accordingly
  • Modularize your code into better structure
  • Host your webpage with Firebase
